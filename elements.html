2. FOOD RATING SYSTEM

BASED ON DATA WAREHOUSE AND MINING

Food Recipe Rating system that detects hidden sentiments in feedback of the customer and rates the feedback accordingly. The system uses opinion mining methodology in order to achieve desired functionality.The System takes feedback of various users, based on the opinion, system will specify whether the posted Food is good, bad, or worst. We use a database of sentiment based keywords along with positivity or negativity weight in database and then based on these sentiment keywords mined in user feedback is ranked.System will use database and will match the feedback with the keywords in database and will rank the feedback.The method Multipe linear regression is used to predict the rating of the food.
PREDICTORS: Calories, Proteins, Fat, Fiber, Sugars, Carbo.
TARGET: Rating.

Internship Project

Head movement controlled Wheel chair

The objective of this project is to construct a wheelchair which can be directly controlled by the head movement.The method employed in this project is the Brain-computer Interface (BCI), which enables direct communication between the head and the electrical wheelchair.
Used Arduino for implementation of the project and Emotiv EPOC head set is used inorder to capture the signals.
some users who completely lost their hands or those who having the difficulty to control their hands such as Poliomyelitis patients are not able to navigate the wheelchair movement. Hence, they are incapable to move around by themselves. With this intention to resolve this issue, another constructive way is by using the brain to directly control the movement of the wheelchair. This method will allow most of the people to navigate the wheelchair by themselves.
